```yml

- Lumberjack
- Miner
- Hunter
- Farmer

- Wood
- Stone
- Food

  Wood:
  - More Housing
  - Higher Wood Cap.
  Stone:
  - Better Tools
  - Higher Stone Cap.

  - More Colonists (Limited By Housing Cap.)
```
